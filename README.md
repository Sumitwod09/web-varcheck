# Varcheck - Software Development Solution

## Project info

A full-stack software development solution based in India. We build mobile apps, websites, and custom software for startups, SMBs, and enterprises worldwide.

## How to run this project

This project uses Node.js and npm. Follow these steps to run the application locally:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
